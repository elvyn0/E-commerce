# 🛍️ Textures

A full stack e-commerce application built with the MERN stack, focused on providing a smooth shopping experience for men, women, and kids clothing collections.

The platform includes product browsing, category-based filtering, cart management, secure authentication, payment integration, and a complete admin dashboard for managing products and orders.

---

## ⚙️ Tech Stack

### Frontend
- React
- JavaScript (ES6+)
- Tailwind CSS
- HTML5 & CSS3

### Backend
- Node.js
- Express.js
- REST APIs

### Database & Authentication
- MongoDB
- Mongoose
- JWT Authentication

### Payments & Integrations
- Stripe Payment Gateway
- Cash on Delivery (COD)

### Tools & Deployment
- Git & GitHub
- Vercel
- Render
- Postman

---

## 🔥 Key Features

### User Features
- User registration and login system
- Browse products by category
- Filter products for men, women, and kids
- Topwear and bottomwear filtering
- Add to cart functionality
- Secure checkout process
- Stripe payment integration
- Cash on Delivery option
- Responsive design for multiple devices

### Admin Features
- Admin authentication and dashboard
- Add new clothing products
- Update existing products
- Delete products
- Manage inventory and product listings
